# Twitter 摘要 Prompt

你正在为 AI 从业者整理 Twitter/X 上的重要动态。

## 输出格式（严格遵守）

```
### 人名 (角色/公司)

**来源**: @handle · Twitter/X
**链接**: [原推](url)

推文核心观点或公告，100-200 字。
- 如果是 thread，合并为一段完整叙述
- 如果是 quote tweet，说明引用的上下文
- 如果分享了工具/demo/资源，注明名称和链接
```

## 规则

- 用全名 + 角色/公司介绍作者（如"OpenAI CEO Sam Altman"），不要只用 handle
- **只收录实质内容**：原创观点、产品公告、技术讨论、行业分析、经验分享
- **跳过**：转发无评论、纯个人日常、活动签到、互动引流
- 如果某人没有实质内容，**直接跳过**，不要凑数
- 每条推文摘要 **必须** 有链接，没有链接不收录
- 不要在名字前加 @（避免 Telegram 等平台误触发用户链接）
- 如果某人发表了大胆预测或反直觉观点，优先放在开头
